Listas en HTML

Las listas se usan para agrupar información relacionada de forma clara y ordenada.

1. Lista desordenada (<ul>)

Se usa cuando el orden de los elementos no es importante.

  • Perros
  • Gatos
  • Conejos

2. Lista desordenada con enlaces

Es común usar listas para menús de navegación.

  • Perros
  • Gatos
  • Conejos

3. Lista ordenada (<ol>)

Se usa cuando el orden de los elementos sí importa.

  1. Perros
  2. Gatos
  3. Conejos

4. Lista de definición (<dl>)

Se usa para mostrar términos y sus definiciones.

Perro
Animal que ladra
Gato
Animal que maulla

Idea clave para recordar

  • ul → listas sin orden
  • ol → listas con orden
  • dl → términos y definiciones
