Títulos y párrafos en HTML

Los títulos organizan el contenido. Los párrafos muestran el texto.

1. Títulos (h1 a h6)

HTML tiene seis niveles de títulos. h1 es el más importante y h6 el menos importante.

h1 — Título principal

h2 — Subtítulo

h3 — Sub-sección

h4 — Nivel inferior

h5 — Poco usado
h6 — Nivel más bajo

Regla básica: una página debe tener un solo h1. Los demás títulos se usan para organizar el contenido.

2. Párrafos (<p>)

La etiqueta <p> se usa para escribir texto en forma de párrafo.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Accusamus ipsam officia laboriosam accusantium ad!

Cada párrafo crea automáticamente un salto de línea antes y después.

3. Saltos de línea (<br>)

La etiqueta <br> se usa para forzar un salto de línea dentro de un mismo párrafo.

Lorem ipsum dolor sit amet,
consectetur adipisicing elit.
Accusamus ipsam officia laboriosam.

Importante: no se debe usar <br> para separar párrafos completos. Para eso se usa <p>.

Idea clave para recordar

  • h1 a h6 organizan el contenido
  • p define párrafos de texto
  • br solo hace un salto de línea
