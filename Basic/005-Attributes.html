Basic HTML Attributes

Atributos globales

Los atributos globales pueden ser utilizados en cualquier elemento HTML:

  • class: Define una o más clases para el elemento.
  • id: Define un identificador único para el elemento.
  • style: Permite aplicar estilos CSS directamente al elemento.

Atributos específicos

Algunos atributos son propios de ciertos elementos. Por ejemplo, en imágenes:

  • src: Especifica la ruta de la imagen a cargar.
  • alt: Proporciona una descripción alternativa si la imagen no se carga.

Atributo title

El atributo title añade información adicional que se muestra al pasar el cursor sobre el elemento.

Ejemplo de uso

<img
src="ruta/a/la/imagen.jpg"
alt="Descripción de la imagen"
title="Foto de ..."
/>

Confusión típica sobre comillas en atributos

Una confusión común es creer que las comillas son siempre obligatorias al definir los valores de los atributos. La verdad es:

  • Para atributos sin espacios, como title, puedes omitir las comillas: 
    <div title=Hola>Texto</div>
  • Sin embargo, si el valor contiene espacios, como en data-attribute, las comillas son necesarias: 
    <div data-info="Valor con espacios">Texto</div>

Atributos booleanos

Si están presentes, se consideran verdaderos. No necesitan valor:

<div hidden>Contenido oculto</div>

Asignar un valor como false no los desactiva. Solo con que existan surten efecto. Son útiles para ocultar elementos y manipularlos con JavaScript.
