Imágenes en HTML

HTML permite mostrar imágenes usando principalmente el elemento img.

El elemento img

El elemento img se usa para insertar una imagen en la página. No tiene etiqueta de cierre. 

<img src="imagen.jpg" alt="Descripción de la imagen">
  • src: ruta del archivo de imagen
  • alt: texto alternativo si la imagen no carga

Ejemplo con imagen SVG

Las imágenes SVG son vectoriales y no pierden calidad al cambiar de tamaño. 

<img src="imagen.svg" alt="Figura circular">

El elemento picture

El elemento picture permite usar diferentes imágenes según el tamaño de la pantalla. 

<picture>
    <source media="(min-width: 1200px)" srcset="imagen-grande.jpg">
    <source media="(min-width: 800px)" srcset="imagen-mediana.jpg">
    <source media="(min-width: 450px)" srcset="imagen-pequena.jpg">
    <img src="imagen.jpg" alt="Imagen de ejemplo">
</picture>

El navegador elige automáticamente la imagen más adecuada.

Qué debes recordar

  • img es el elemento principal para imágenes
  • alt siempre debe estar presente
  • picture se usa cuando necesitas imágenes adaptables
