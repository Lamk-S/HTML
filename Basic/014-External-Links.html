Enlaces a otros sitios web

Los enlaces externos llevan al usuario fuera de tu sitio web.

Ejemplo básico

Un enlace externo apunta a una URL completa:

  • Visitar W3Schools

Abrir en una nueva pestaña

Para no sacar al usuario de tu sitio, puedes abrir el enlace en otra pestaña.

  • Visitar MDN Web Docs

Qué debes recordar

  • Un enlace externo usa una URL completa (https://)
  • target="_blank" abre el enlace en otra pestaña
  • rel="noopener noreferrer" mejora la seguridad
