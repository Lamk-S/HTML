Formato de texto en HTML

HTML permite dar significado y énfasis al texto usando distintas etiquetas.

Etiquetas semánticas (recomendadas)

Estas etiquetas indican qué significa el texto, no solo cómo se ve.

Texto importante → <strong>

Texto enfatizado → <em>

Texto marcado / resaltado → <mark>

Texto más pequeño → <small>

Texto borrado → <del>

Texto insertado → <ins>

Subíndices y superíndices

Se usan comúnmente en fórmulas matemáticas o químicas.

H2O → <sub>

x2 → <sup>

Etiquetas solo visuales (uso limitado)

Estas etiquetas no aportan significado, solo cambian la apariencia.

Texto en negrita → <b>

Texto en itálica → <i>

Recomendación: prioriza <strong> y <em> en lugar de <b> y <i>.

Idea clave para recordar

  • HTML no es solo visual, también es semántico
  • strong y em indican importancia y énfasis
  • b e i solo cambian el aspecto
