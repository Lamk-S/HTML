Sintaxis básica de HTML

HTML se basa en etiquetas que indican al navegador cómo mostrar el contenido.

1. ¿Qué es un elemento HTML?

Un elemento HTML normalmente está compuesto por:

  • Una etiqueta de apertura
  • Contenido
  • Una etiqueta de cierre
<elemento>
    Contenido
</elemento>

Ejemplo real:

<p>Hola mundo!</p>

Resultado en el navegador:

Hola mundo!

2. Etiqueta de apertura y cierre

  • La etiqueta de apertura: <p>
  • La etiqueta de cierre: </p>

La barra / indica el final del elemento.

3. Comentarios en HTML

Los comentarios sirven para explicar el código y no se muestran en el navegador.

<!-- Este es un comentario en HTML -->

Son útiles para estudiar, documentar o trabajar en equipo.

4. Atributos HTML

Los atributos agregan información extra a las etiquetas.

Siempre se colocan en la etiqueta de apertura.

<elemento atributo="valor">
</elemento>

Ejemplo práctico:

<p id="myID" class="myClass">Atributos!</p>

Explicación:

  • id → identifica un elemento único
  • class → agrupa elementos para estilos o JavaScript

5. Etiquetas más comunes

  • <p> → párrafos
  • <h1>...<h6> → títulos
  • <a> → enlaces
  • <img> → imágenes
  • <div> → contenedor genérico

Existen muchas más etiquetas; pueden consultarse en la documentación oficial o en sitios educativos como W3C y W3Schools.

Idea clave para recordar

  • HTML usa etiquetas para estructurar contenido
  • Las etiquetas pueden tener atributos
  • El navegador interpreta HTML, no lo ejecuta
