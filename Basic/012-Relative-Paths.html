Rutas relativas en HTML

Las rutas relativas permiten enlazar archivos dentro del mismo proyecto.

Ejemplo básico

Desde esta página puedes navegar a otros recursos:

  • Ir a otra página
  • Enviar un correo
  • Enlace pendiente

Qué debes recordar

  • Si el archivo está en la misma carpeta, basta con el nombre: otraPagina.html
  • mailto: se usa para abrir el correo del usuario
  • # se usa como enlace temporal mientras se desarrolla
