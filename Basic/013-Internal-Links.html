Enlaces a otras páginas del sitio

Los enlaces internos permiten navegar entre las páginas de tu propio sitio web.

Estructura del proyecto

Este ejemplo asume la siguiente estructura:

folder/
├─ index.html
├─ pages/
│ ├─ about.html
│ └─ contact.html
└─ assets/

Ejemplos de enlaces

  • Ir a About
  • Ir a Contact

Uso de ../

Cuando un archivo está en una carpeta superior, se usa ../ para subir un nivel.

Ejemplo desde pages/about.html para volver al inicio: 

<a href="../index.html">Volver al inicio</a>

Errores comunes

  • Escribir mal el nombre del archivo
  • Olvidar la carpeta donde está el archivo
  • Usar mayúsculas diferentes en rutas
