# Online Course – HTML Basic Structure

Este proyecto demuestra la **estructura HTML básica de un caso real**:
una página web de un curso online.

## Características
- Uso correcto de etiquetas semánticas
- Jerarquía clara de encabezados
- Secciones reales usadas en producción
- Formularios y contenido estructurado

## Tecnologías
- HTML5

## Enfoque
Este proyecto se centra únicamente en **estructura y semántica**.
No incluye CSS ni JavaScript de forma intencional.
