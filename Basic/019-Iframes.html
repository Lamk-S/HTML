Atributos importantes

  • src: URL del contenido a mostrar
  • width / height: tamaño del iframe
  • name: permite usar target desde enlaces
  • frameborder: define el borde (obsoleto, pero común)

Qué debes recordar

  • No todos los sitios permiten ser cargados en iframes
  • El iframe muestra contenido externo, no lo copia
  • Se usa para mapas, videos o herramientas externas
