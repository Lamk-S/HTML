# Apuntes del Curso de HTML

Este repositorio contiene mis apuntes del curso de **HTML**, organizados para consulta rápida y práctica.

## Herramientas usadas
- **Visual Studio Code** como editor principal.
- Extensión **Live Preview** para visualizar los cambios en tiempo real mientras escribo el código.

## Objetivo
Mantener un registro claro y ordenado de todo lo aprendido en el curso, así como practicar la estructura y semántica correcta en HTML.
